Bài 3 cách tải và sử dụng bootstrap framework

Hướng dẫn tải, cài đặt và sử dụng:
B1: vào trang getbootstrap để tải phiên bản bootstrap cần thiết(bootstrap gồm các file css, js được viết sẵn)
B2: link thư viện bootstrap vào trang html
B3: chỉ cần khai báo các class vào các phần tử trang html, vì bootstrap là những đoạn code được viết sẵn
